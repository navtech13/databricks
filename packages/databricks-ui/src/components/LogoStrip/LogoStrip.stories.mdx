## LogoStrip

`<LogoStrip />` is a component that displays a strip of logos.

### Props

This componen receives a custom array of images as prop:

<Props of={LogoStrip} />

<Preview>
  <Story name='Logo Strip'>
    {(args) => {
      const [logos, setLogos] = useState([])
      useEffect( () => {
        const asyncFn = async () => {
        const response = await fetch(`/logos`)
        const { data } = await response.json()
        setLogos(data.logos)
        }
        asyncFn();
      }, [])
      return <LogoStrip logos={logos.slice(0, args.logos)} />
    }}
  </Story>
</Preview>
