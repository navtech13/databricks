# Multi Media
Multi Media component allows render different types of multimedia content, such as images, videos, and animations with the follow formats: PNG, JPG, GIF, SVG or JSON animation.
The component accepts various properties to customize its behavior and appearance such as: alignment, background, columns, motion control, padding bottom and top.

              motionControl

          />
        )
      }
